# 💯 100Apps

> **100個のアプリを作る個人プロジェクト**

## 概要

このリポジトリは、100個のアプリケーションを作ることを目標とした長期プロジェクト
主な目的は、研究の合間でも継続的にコードを書き、Pythonのコーディングスキルを維持・向上させること
AIによるコーディング支援が一般的になった現在でも、自分で設計し、自分で実装できる力を維持するのが目的
完成度の高いアプリを作ることよりも、「実際に手を動かして学ぶこと」を大切に

---

## 目標

- 様々な言語でアプリを100個作る
- コーディングスキルを維持・向上させる
- 新しいライブラリや技術に触れる
- 思いついたアイデアをすぐ形にする
- 楽しみながら開発を続ける

---

## AIとの付き合い方

このリポジトリでは、プロジェクトごとにAIの利用方針が異なります。

### AIを活用するプロジェクト

- 新しいライブラリの学習
- 開発効率の向上
- AIとの協調開発の練習
- 研究で使いそうなライブラリがあれば、触ってみる

### AIを使わない（または最小限にする）プロジェクト

- Pythonの基礎力を維持する
- アルゴリズムや実装力を鍛える
- 自分の手でコードを書く習慣を保つ

どちらが目的というわけではなく、そのプロジェクトで学びたいことに応じて使い分ける
