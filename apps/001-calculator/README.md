# 001 Calculator

## 概要

Tkinterを用いて四則演算ができる電卓アプリを作成する。

## 使用技術

- Python
- Tkinter

## AI

**Teacher**

文法調査・エラー調査のみ利用。ほぼ未使用

## メモ

- Tkinterを用いたGUIアプリケーションの基本的な作成方法
- `grid()` を用いたレイアウト管理
- `Button`、`Entry`などの基本ウィジェットの使い方
- `command` を利用したイベント処理
- `lambda` を用いてボタンごとに異なる値を関数へ渡す方法
- `Entry.get()`、`Entry.delete()`、`Entry.insert()` を利用した画面の更新方法
- `enumerate()` を利用してボタンをループで生成する方法
- `//` と `%` を利用した2次元グリッドへの配置方法
- `eval()` を利用した簡単な数式の計算方法
- GUIアプリでは、画面表示とロジックを関数ごとに分けることで実装しやすくなること
- 難易度：簡単
